# Estate - Contractor & Real Estate Website

Estate is a modern, responsive contractor and real estate website built using **React.js** and **Tailwind CSS**. It features smooth animations, a clean UI, project showcases, testimonials, and a fully working contact form.

## Features
- Fully responsive contractor & real estate website  
- Smooth animations using Framer Motion  
- Projects slider  
- Testimonials section  
- Working contact form with Web3Forms  
- Clean and modern UI design  

## Technologies Used

**Frontend**
- React.js – Component-based UI  
- Tailwind CSS – Responsive styling  
- Framer Motion – Animations  

**Form Handling**
- Web3Forms – Contact form submissions  

## Installation
1. Clone the repository  
2. Navigate to the project folder  
3. Install dependencies using `npm install`  
4. Start the development server using `npm run dev`
